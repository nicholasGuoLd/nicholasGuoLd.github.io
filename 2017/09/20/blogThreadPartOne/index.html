Thread knowledge:

  2. Achieve the port of “Runnable” to begin one thread
Tips: The ability of extending in inheritance is not so strong and Java always support single thread, which means if one class inherit “Thread” class, it cannot inherit anymore classes.

How to run the thread?

Please use “start” method to run the thread, however, after running it should use “run” method to call “Thread” class.
